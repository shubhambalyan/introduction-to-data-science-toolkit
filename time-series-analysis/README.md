# Time Series Analysis in R

* Install R packages: 
  > install.packages("faraway") 
* Access data in a package: 
  > data(package='faraway')
* Load a dataset: 
  > data(coagulation, package='faraway')
* List the loaded datasets: 
  > ls()
* Show the dataset: 
  > dataset-name ex: "coagulation"
* Summary of dataset:
  > summary(coagulation)
